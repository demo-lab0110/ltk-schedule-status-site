# LTK Schedule & Status サイト設計

この文書は、LTK Schedule & Status サイト本体の最低限の設計書である。

## 目的

LTK Schedule & Status は、League The k4sen 関連の予定、結果、配信状況、スタッツを表示するための静的サイトである。

サイトは表示専用であり、スプレッドシートへは書き込まない。

## 既存の主な機能

- 予定
- リーグ表
- チーム戦績
- 個人成績
- 個人ランキング
- チャンピオン
- 切り抜き動画
- Draft Predictor
- このサイトについて
- お問い合わせ

## 既存構成

- `index.html` にタブボタンと各画面のsectionが定義されている。
- `app.js` がタブクリック、表示切替、データ描画を担当する。
- 現在はタブUIで各画面を切り替えている。
- 現時点では、URLごとにタブ状態を直接表すルーティングは実装していない。

## URL付きタブ化の方針

今後、各タブへ直接アクセスできるようにURL付きタブ化を予定する。

URL付きタブ化では、既存UIと既存タブ挙動を壊さないことを優先する。

GitHub Pages等の静的配信で404を避けるため、まずはHashルーティングを基本方針とする。

基本形式:

```text
/#/schedule
/#/standings
/#/teams
/#/players
/#/rankings
/#/champions
/#/about
/#/contact
```

対応:

- `/#/schedule` は予定タブを開く。
- `/#/standings` はリーグ表タブを開く。
- `/#/teams` はチーム戦績タブを開く。
- `/#/players` は個人成績タブを開く。
- `/#/rankings` は個人ランキングタブを開く。
- `/#/champions` はチャンピオンタブを開く。
- `/#/about` はこのサイトについてタブを開く。
- `/#/contact` はお問い合わせタブを開く。

`/#/draft` は今回のURL付きタブ化の実装対象外とし、将来予約ルートとして扱う。現時点で対応するタブやsectionを追加しない。

不明なHashは安全に既定タブへ戻す。既定タブは `/#/schedule` とする。

## 切り抜き動画ページの公開方針

切り抜き動画ページは本番公開対象とする。

既存実装に `CLIPS_PREVIEW_ENABLED` のようなローカル限定フラグがある場合、本番でも表示できるようにする。

切り抜き動画ページ内には、YouTubeとTwitch Clipsの2つのサブタブがある。

- `/#/clips` にアクセスした場合は、YouTubeタブを初期表示する。
- `/#/clips/youtube` にアクセスした場合は、YouTubeタブを表示する。
- `/#/clips/twitch` にアクセスした場合は、Twitch Clipsタブを表示する。
- `/#/clips/unknown` のような不正なclips配下URLの場合は、`/#/clips` にフォールバックする。

YouTube/TwitchのサブタブURL対応は、既存の切り抜き動画UIを壊さず、既存のタブ表示状態と同期する。
